== Installation ==

**There are 2 ways to install Easy Call To Action Buttons on your blog:**

Upload the plugin to your blog plugin directory using ftp client or web file manager and activate it or;

Visit your blog admin panel and:

1. Click Plugins in the menu
2. Click Add New
3. In search field type "Easy Call To Action Buttons" (without quotes)
4. Find Easy Call To Action Buttons in the list and click Install Now.
